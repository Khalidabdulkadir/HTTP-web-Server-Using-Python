# HTTP Web Server Using Python

This project is a simple HTTP web server implemented in Python. It is designed to handle basic HTTP requests and serve static files.

## Features

- Handles GET and POST requests
- Serves static files (HTML, CSS, JS, images)
- Basic routing capabilities
- Simple logging of requests

## Requirements

- Python 3.x

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/Khalidabdulkadir/HTTP-web-Server-Using-Python
    ```
2. Navigate to the project directory:
    ```sh
    cd Http Web server Using Python
    ```

## Usage

1. Run the server:
    ```sh
    python server.py
    ```
2. Open your web browser and navigate to `http://http://127.0.0.1:8080`.
